Developed a custom Matrix class in Java implementing core linear algebra operations and matrix property checks. The project supports matrix creation from both in-memory arrays and external text files, matrix addition, scalar multiplication, matrix multiplication with dimension validation, and identification of diagonal, upper triangular, and lower triangular matrices. A comprehensive test suite was created to verify functionality, demonstrate object-oriented design principles, exception handling, file I/O, and multidimensional array manipulation in Java.

Technologies & Concepts Demonstrated:

Java
Object-Oriented Programming (OOP)
File I/O using BufferedReader and FileReader
Exception Handling
Multidimensional Arrays
Matrix Algebra Algorithms
Input Validation and Error Handling
Unit-style Functional Testing
